# Graph Traversal Project

This project implements breadth-first traversal and depth-first traversal for a directed graph.

The graph uses vertices A through I and follows the edges provided in the assignment.

## Features

- Directed graph implementation
- Breadth-first traversal using a queue
- Depth-first traversal using a stack
- BFS traversal tree
- DFS traversal tree

## Expected BFS Output

Starting from vertex A:

A B D E G F H C I

## Expected DFS Output

Starting from vertex A:

A B E F C H I D G
